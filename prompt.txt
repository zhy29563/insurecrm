请基于 Flutter 3.x + Android Studio，生成一个**完整可直接运行**的保险客户与产品管理系统 APP。
项目用途：保险从业者专用，管理客户信息 + 保险产品资料。

你需要一次性生成**全部完整代码**，包括：
- 完整项目结构
- pubspec.yaml
- lib 下所有代码文件
- 所有页面、模型、工具类
- 本地数据存储（无需后端，无需复杂第三方依赖）
- 可直接运行、直接打包

============================
【项目功能要求】
============================
一、客户信息管理
1. 姓名（支持别名）
2. 年龄、性别
3. 多个手机号
4. 多个地址
5. 产品购买历史记录
6. 多次拜访记录
7. 客户评级（购买意向）
8. 支持按姓名、手机号、地址搜索
9. 地图定位客户位置
10. 按位置搜索附近客户（可选择半径）
11. 客户关联功能（家人/同事）
12. 添加合作同事信息

二、保险产品管理
1. 产品所属公司
2. 产品内容介绍
3. 产品优势、产品分类
4. 产品生效日期、结束日期
5. 同公司产品对比
6. 跨公司同类产品对比
7. 根据客户信息智能推荐产品

三、界面要求
1. 包含：登录页、首页、客户列表页、产品列表页、设置页
2. UI 美观、简洁、专业、适合保险从业者使用
3. 自适应手机、平板、电脑
4. 支持深色/浅色模式

四、跨平台要求
1. 支持：Android、iOS、鸿蒙、Windows、Linux、macOS、Web
2. 纯 Flutter 原生实现，不引入复杂第三方依赖
3. 本地数据存储，无需网络、无需后端

============================
【必须输出内容】
============================
1. 完整项目目录结构
2. 所有代码文件（一次性全部给出）
3. 运行命令
4. 各平台打包命令
5. 各平台安装方法
6. 无需我修改任何代码，复制即可使用
